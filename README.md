# Paper Base Duel Releases

This repository hosts public release packages for **Paper Base Duel / 纸上基地**.

The first public build target is **Public Alpha 0.5.0**. Release packages will be attached to GitHub Releases in this repository.

## About The Game

Paper Base Duel is a two-player grid-paper strategy game built from a paper prototype. Public Alpha 0.5.0 includes the paper-style interface, local two-player mode, player-vs-AI mode, mainline tutorials, localized text, shop and economy systems, feedback effects, and siege tools.

## Download

Use the latest GitHub Release on this repository and download the package for your platform.

Expected first-release packages:

- `PaperBaseDuel_0_5_0_macOS.zip`
- `PaperBaseDuel_0_5_0_windows.exe`

Additional platforms may be added later if they can be tested.

## Repository Scope

This repository is release-only. It does not contain source code, development history, or editable source assets.

Public packages are exported builds only. Credits and license information should be distributed with the release package and mirrored in this repository when applicable.

## Status

Public Alpha builds are playtest builds. Rules, balance, UI, and content may change in later versions.
